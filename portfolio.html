Projects

Below is the Crystal Collector! Each crystal is assigned a hidden random value. You must match your total to the number displayed in order to win. The languages used to develop this are HTML, CSS, and JavaScript. Click the crystals to increase your number, but make sure not to go too high or you will lose. Click Me to Play!

Below is the Hominid Trivia Game. Just like the title suggests, it is a game that tests your knowledge about Hominid facts. Utilizing HTML, CSS, and JavaScript, This app is also mobile responsive to all screen dimensions with simple clean code. Click Me to Play!

Below app is the Resturant Rater. It is designed to let a user write an anonymous review about a resturant. They can also create a resturant to review if one is not on the list. Also they can edit, delete, and view other reviews. The core technologies used in this app are Express, MySQL, bootstrap, and Sequelize, while also using HTML, CSS, and JavaScript languages. Click Me to Play!

Below is the Burger Maker. To run this application, first you will have to npm install express, express-handlebars, mysql, and sequelize. This application is designed to let a user create a burger with a name and a state of devoured or fresh. Once the burger is made it will be put into either category on the screen. If the user wishes to change the state of the burger they can simply click on the button. Click Me to Play!

Below is the Dragonball Hangman game. This is a traditional hangman game with character names from the Dragonball Z franchise. Although there are no instructions provided, the game is pretty intuitive and is fun to play. This application runs on HTML, CSS, JavaScript. Click Me to Play!

Team-Built Applications

Below is the Movie Cheat Sheet App. This program is designed to deliver Movie or Television information relevant to a user search. Movie Cheat Sheet runs by using an API query to bring the user what they are looking for. Along with a gify, a poster will appear for anything related to the search as well as ratings, cast, and other information for specified movies or shows. This is also mobile responsive across all devices using media queries. Click Me to Play!

Below is KnodeyaCode. An application built with team members for our final project in class. There are a selection of games and different functions all using React.js as a single page application. This application uses a mix of bootstrap and home-made CSS to give a clean professional layout. Click Me to Play!
